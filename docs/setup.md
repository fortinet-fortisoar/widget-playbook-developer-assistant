# Installation
1. To install a widget, click **Content Hub** > **Discover**.
2. From the list of widgets that appears, search **Playbook Developer Assistant** widget.
3. Click to open the **Playbook Developer Assistant** widget card.
4. Click **Install** on the lower part of the screen to begin installation.

# Configuration


> [!NOTE]  
> The **Playbook Developer Assistant** launches automatically when a playbook is opened in the playbook designer, and requires no additional configuration.
